Reach Enumerator

Reach is an enumerator for the Alloy modeling language that lets the user view scenarios by size.

Reach Information

Item Description
Tool Latest stable release of Reach which is a self-executable JAR file.
Source Code Open source code base for the latest release. Includes a collection of models which can be used to explore Reach.
Requirements Reach has the following requirements for execution:
  • Supported Operating Systems: Linux (64 bit) and Mac OS X (64 bit).
  • Java 8: Must be installed and accessible from PATH.
